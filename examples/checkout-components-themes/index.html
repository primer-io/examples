Custom-Styled Checkout

This example demonstrates the power of Composable Checkout's style tokens by allowing you to toggle between eight different visual themes. The implementation uses pure HTML with a minimal TypeScript script and relies primarily on CSS variables.

Available Themes:

  • Light Theme: The standard Primer checkout styling
  • Dark Theme: A dark-themed checkout for low-light environments
  • Minimal: A clean, simplified design with subtle styling
  • High Contrast: Enhanced visual contrast for better accessibility
  • Kawaii: A cute, colorful design with playful elements
  • Brutalist: Raw, minimal design with strong contrast and bold elements
  • Neon Cyberpunk: Vibrant neon colors on dark backgrounds with futuristic styling
  • Pimp My Checkout: Flashy, chrome-like styling inspired by the TV show "Pimp My Ride" (Easter egg theme)
